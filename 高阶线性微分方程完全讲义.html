SECTION 01

高阶线性方程·解的结构

1.1 方程的一般形式

n 阶线性微分方程的标准形式（已将最高阶系数化为 1）：

y(n) + a1(x)y(n−1) + ⋯ + an−1(x)y′ + an(x)y = f(x)     (*)

其中 a1(x), …, an(x) 称为变系数（都是已知函数），f(x) 是非齐次右端项。

非齐次方程 (*)

f(x) ≠ 0 时，(*) 称为非齐次线性方程。

其解集记为 T。

对应的齐次方程 (**)

将右边换成 0：

y(n) + a1(x)y(n−1) + ⋯ + an(x)y = 0   (**)

其解集记为 S。

⚠ 关键背景

n ≥ 2 的高阶方程没有通解公式（不像一阶线性方程有积分因子公式）。因此我们转而研究解的结构，把问题拆成两步：① 找齐次方程的 n 个独立解；② 找非齐次方程的一个特解。

1.2 解集的三条性质（核心定理）

定理：T 与 S 之间的关系

设 T 是非齐次方程 (*) 的解集，S 是齐次方程 (**) 的解集：

性质 (1) — S 对线性组合封闭

若 y₁, y₂ ∈ S，则对任意常数 k, l：

ky₁ + ly₂ ∈ S

齐次解的任意线性组合仍是齐次解。

性质 (2) — 两非齐次解之差 ∈ S

若 y₁, y₂ ∈ T，则：

y₁ − y₂ ∈ S

两个非齐次解相减，变成齐次解。

性质 (3) — 齐次解 + 特解 ∈ T

若 y₁ ∈ S，y₂ ∈ T，则：

y₁ + y₂ ∈ T

齐次解 + 非齐次特解 = 非齐次解。

证明思路（三条性质的统一逻辑）

把相关函数代入方程，对方程做线性运算（乘以常数、相加减）：由于方程是线性的，线性运算可以穿过方程算子，右边自动得到 0（齐次）或 f(x)（非齐次）。例如性质(1)：把 ky₁ + ly₂ 代入 (**) 左边 = k·[(**) 对 y₁] + l·[(**) 对 y₂] = k·0 + l·0 = 0。

1.3 唯一性定理与通解结构

唯一性定理

若 y(x) 满足齐次方程 (**) 且在某点 x₀ 处所有初值全为零：

y(x₀) = y′(x₀) = y″(x₀) = ⋯ = y(n−1)(x₀) = 0

则 y(x) ≡ 0（恒为零函数）。

推论：齐次方程通解 — Wronskian 行列式条件

若 y₁(x), y₂(x), …, yₙ(x) 是 (**) 的 n 个解，且在某点 x₀ 处 Wronskian 行列式不为零：

y₁(x₀)y₂(x₀)⋯yₙ(x₀)
y₁′(x₀)y₂′(x₀)⋯yₙ′(x₀)
⋮⋮⋱⋮
y₁(n−1)(x₀)y₂(n−1)(x₀)⋯yₙ(n−1)(x₀)
≠ 0

则 (**) 的任意解都可以写成：

y(x) = C₁y₁(x) + C₂y₂(x) + ⋯ + Cₙyₙ(x)

此时称 y₁, …, yₙ 是 (**) 的 n 个线性独立的解（基本解组）。

最终结论：非齐次方程通解公式

若 y*(x) 是非齐次方程 (*) 的任意一个特解，则 (*) 的通解为：

y(x) = C₁y₁(x) + C₂y₂(x) + ⋯ + Cₙyₙ(x) + y*(x)

= 齐次通解 + 非齐次特解

📌 好消息与坏消息（课件原话）

好消息 只要找到齐次方程的 n 个独立解 + 非齐次方程的一个特解，通解就全解决了。

坏消息 对变系数方程，通常找齐次方程的哪怕 1 个解都很难！特解更难。

好消息 对常系数方程（下面要学的），有系统方法！

SECTION 02

二阶常系数齐次方程

2.1 方程形式与核心思路

设 p, q 为常数，考察：

y″(x) + py′(x) + qy(x) = 0     (**)
关键思路：试探解 y = eλx

将 y = eλx 代入 (**)，由于求导只带来一个因子：

(eλx)″ + p(eλx)′ + q·eλx = λ²eλx + pλeλx + qeλx = eλx(λ² + pλ + q)

因为 eλx ≠ 0，所以 y = eλx 是解的充要条件是括号内为零：

λ² + pλ + q = 0    （特征方程）

这是一个关于 λ 的二次代数方程，判别式 Δ = p² − 4q。

2.2 三种情形的通解

Δ > 0 — 特征方程有两个不等实根 λ₁ ≠ λ₂

取两个线性独立的解：

y₁(x) = eλ₁x,    y₂(x) = eλ₂x

通解：

y(x) = C₁eλ₁x + C₂eλ₂x
例  |  y″ − 5y′ + 6y = 0

特征方程：λ² − 5λ + 6 = 0

解：(λ−2)(λ−3) = 0，得 λ₁ = 2，λ₂ = 3，Δ = 25−24 = 1 > 0 ✓

y = C₁e2x + C₂e3x

Δ = 0 — 特征方程有重根 λ₁ = λ₂ = λ

此时只有一个独立的指数解 y₁ = eλx。如何找第二个独立解？

推导第二个解（极限法）

当 λ₂ → λ₁ 时：

y₂(x) = limλ₂→λ₁ (eλ₂x − eλ₁x)/(λ₂ − λ₁) = xeλx

（这相当于对 eλx 关于 λ 求导，本质是导函数仍满足方程。）可以直接代入验证 xeλx 也是解。

通解：

y(x) = (C₁ + C₂x)eλx
⚠ 重根时的错误

错 y = C₁eλx + C₂eλx = (C₁+C₂)eλx（本质只有一个常数！）

对 第二个解必须是 xeλx，两者乘以 x 才真正独立。

例  |  y″ − 8y′ + 16y = 0

特征方程：λ² − 8λ + 16 = 0

解：(λ−4)² = 0，得 λ₁ = λ₂ = 4，Δ = 64−64 = 0 ✓

y = (C₁ + C₂x)e4x

Δ < 0 — 特征方程有共轭复根 λ₁,₂ = a ± bi （b ≠ 0）

推导实值解（欧拉公式）

复数形式的解 e(a±ib)x 代入欧拉公式 eiθ = cos θ + i sin θ：

e(a±ib)x = eax(cos bx ± i sin bx)

取实部和虚部，得到两个实值线性独立解：

y₁(x) = eaxcos bx,     y₂(x) = eaxsin bx

通解：

y(x) = eax(C₁cos bx + C₂sin bx)
⚠ 注意 a 和 b 的来源

复根 λ = a ± bi，其中 a = −p/2（实部），b = √(4q−p²)/2（虚部）。

特别地，若 p = 0，则 a = 0，通解变为 y = C₁cos bx + C₂sin bx（纯正弦振荡）。

例  |  y″ − 6y′ + 25y = 0

特征方程：λ² − 6λ + 25 = 0

解：λ = (6 ± √(36−100))/2 = (6 ± √(−64))/2 = 3 ± 4i

所以 a = 3，b = 4，Δ = 36−100 = −64 < 0 ✓

y = e3x(C₁cos 4x + C₂sin 4x)

2.3 三情形速查

判别式 Δ = p²−4q特征根两个独立解通解形式
Δ > 0 λ₁ ≠ λ₂（两实根） eλ₁x，eλ₂x C₁eλ₁x + C₂eλ₂x
Δ = 0 λ₁ = λ₂ = λ（重根） eλx，xeλx (C₁ + C₂x)eλx
Δ < 0 λ = a ± bi（复根） eaxcos bx，eaxsin bx eax(C₁cos bx + C₂sin bx)
SECTION 03

待定系数法求特解

3.1 总体框架

对二阶常系数非齐次方程：

y″(x) + py′(x) + qy(x) = f(x)     (*)

找到特解 y*(x) 后，通解 = 齐次通解 + y*。核心思路：

观察 f(x) 的形式
→
查 λ₀ 是否是特征根（及重数）
→
确定 y* 的"模板"形式
→
代入方程，比较系数
⚠ 第一步必须：检查 λ₀ 与特征根的关系！

每一种情形，在设特解之前，必须先解出特征方程的根 λ₁, λ₂，然后比较 f(x) 中隐含的 λ₀ 与它们的关系。根的重数直接决定特解模板中要额外乘几次 x。

3.2 五种情形 — f(x) = eλ₀xPₙ(x) 型

这四种情形都属于 f(x) = eλ₀xPₙ(x) 的大类（Pₙ 是 n 次多项式），区别仅在于 λ₀ 是不是特征根及重数。

情形 1 f(x) = eλ₀xPₙ(x)，且 λ₀ 不是特征根（k=0）

设特解：

y*(x) = eλ₀xQₙ(x)

其中 Qₙ 是同次数（n次）的待定多项式，代入方程，比较各次幂系数，解出 Qₙ 的系数。

例  |  y″ − 2y′ − 3y = 2x² + 1  （λ₀ = 0，Pₙ = 2x²+1 为2次多项式）

Step 1：解特征方程 λ²−2λ−3=0，λ₁,₂ = −1, 3。

λ₀ = 0 不是特征根（−1 ≠ 0，3 ≠ 0），用情形1。

Step 2：设 y* = ax² + bx + c（同为2次多项式，e0·x=1）。

Step 3：代入：y*″ = 2a，y*′ = 2ax+b

代入方程：2a − 2(2ax+b) − 3(ax²+bx+c) = 2x²+1

展开：−3ax² + (−4a−3b)x + (2a−2b−3c) = 2x²+1

Step 4：比较系数：
x² 项：−3a = 2 → a = −2/3
x¹ 项：−4a−3b = 0 → b = −4a/3 = 8/9
x⁰ 项：2a−2b−3c = 1 → c = (2a−2b−1)/3 = −37/27

y = C₁e−x + C₂e3x − (1/27)(18x² − 24x + 37)
情形 2 f(x) = eλ₀xPₙ(x)，且 λ₀ 是特征方程的单实根（k=1，Δ > 0）

λ₀ 已经是一个特征根，原来的"eλ₀x·多项式"形式已经被齐次通解包含了，需要额外乘一个 x：

y*(x) = eλ₀x · x · Qₙ(x)
例  |  y″ − 2y′ = 2x  （λ₀ = 0，Pₙ = 2x 为1次多项式）

Step 1：λ²−2λ=0，λ₁=0，λ₂=2。

λ₀ = 0 等于 λ₁，是单根，用情形2。

Step 2：设 y* = x(ax+b)（x·一次多项式 = 二次）。

Step 3：y*′ = 2ax+b+x(2a) = wait… 直接展开：y* = ax²+bx，y*′=2ax+b，y*″=2a

代入：2a − 2(2ax+b) = 2x → 2a−2b−4ax = 2x

Step 4：x¹：−4a=2 → a=−1/2；x⁰：2a−2b=0 → b=a=−1/2

y = C₁ + C₂e2x − (1/2)x(x+1)
例  |  y″ + 5y′ − 6y = xex  （λ₀ = 1，Pₙ = x 为1次）

Step 1：λ²+5λ−6=0，λ₁=−6，λ₂=1。

λ₀ = 1 等于 λ₂，是单根，用情形2。

Step 2：设 y* = x(ax+b)ex。

Step 3：展开代入（过程繁琐），化简后：14ax + (2a+7b) = x

Step 4：14a=1 → a=1/14；2a+7b=0 → b=−2a/7=−1/49

y = C₁e−6x + C₂ex + x(x/14 − 1/49)ex
情形 3 f(x) = eλ₀xPₙ(x)，且 λ₀ 是特征方程的二重实根（k=2，Δ = 0）

λ₀ 是二重根，需要额外乘 x²：

y*(x) = eλ₀x · x² · Qₙ(x)
例  |  y″ − 2y′ + y = ex  （λ₀ = 1，Pₙ = 1 为0次多项式）

Step 1：λ²−2λ+1=0，(λ−1)²=0，λ₁=λ₂=1。

λ₀ = 1 是二重根，用情形3。

Step 2：设 y* = x²·a·ex（x²·0次多项式）。

Step 3：代入方程，化简得 2a = 1 → a = 1/2。

y = (C₁ + C₂x)ex + (1/2)x²ex
🔑 统一规律（实根三情形合并记忆）

f(x) = eλ₀xPₙ(x) 时，特解模板：

y*(x) = eλ₀x · xk · Qₙ(x)

其中 k = λ₀ 作为特征根的重数（不是根则 k=0，单根则 k=1，二重根则 k=2）。Qₙ 与 Pₙ 次数相同。

3.3 三角函数型 f(x)

当 f(x) 含三角函数时，属于以下两种情形。

情形 4 f(x) = ea₀x(P¹ₙcos b₀x + P²ₙsin b₀x)，且 λ₀ = a₀+ib₀ 不是特征复根（k=0）

设特解：

y*(x) = ea₀x(Q¹ₙ(x)cos b₀x + Q²ₙ(x)sin b₀x)

注意：即使 f(x) 只有 cos 或只有 sin，特解中两者都要设！（因为求导会交叉产生另一个。）

例  |  y″ + y = sin 3x  （a₀=0，b₀=3，故 λ₀ = 3i）

Step 1：λ²+1=0，λ₁,₂ = ±i。

λ₀ = 3i 不等于 ±i，不是特征根，用情形4。

Step 2：设 y* = a cos 3x + b sin 3x（Q¹₀=a，Q²₀=b，均为0次多项式）。

Step 3：y*″ = −9a cos 3x − 9b sin 3x，代入：

(−9a+a)cos 3x + (−9b+b)sin 3x = sin 3x → −8a cos 3x − 8b sin 3x = sin 3x

Step 4：cos 3x 系数：−8a=0 → a=0；sin 3x 系数：−8b=1 → b=−1/8

y = C₁cos x + C₂sin x − (1/8)sin 3x
情形 5 f(x) = ea₀x(P¹ₙcos b₀x + P²ₙsin b₀x)，且 λ₀ = a₀+ib₀ 是特征复根（k=1）

λ₀ 已是特征根，额外乘一个 x：

y*(x) = ea₀x · x · (Q¹ₙ(x)cos b₀x + Q²ₙ(x)sin b₀x)
例  |  y″ + y = 2cos x  （a₀=0，b₀=1，故 λ₀ = i）

Step 1：λ²+1=0，λ₁,₂ = ±i。

λ₀ = i 等于 λ₁，是特征根！用情形5。

Step 2：设 y* = x(c cos x + d sin x)。

Step 3：求导：y*′ = (c cos x + d sin x) + x(−c sin x + d cos x)

y*″ = 2(−c sin x + d cos x) + x(−c cos x − d sin x)

代入 y*″ + y*：= 2(−c sin x + d cos x) + x(−c cos x − d sin x) + x(c cos x + d sin x)

= 2(−c sin x + d cos x) + x·0 = 2d cos x − 2c sin x = 2cos x

Step 4：cos x 系数：2d=2 → d=1；sin x 系数：−2c=0 → c=0

y = C₁cos x + C₂sin x + x sin x
⚡ 物理直觉：共振现象！

y″ + y = 0 描述一个固有频率为 1 的振动系统（弹簧）。当外力 2cos x 的频率恰好等于固有频率（即 λ₀ = ±i 正好是特征根）时，产生共振——特解中出现 x sin x，振幅随时间线性增大！这就是情形5 中额外乘 x 的物理含义。

🔑 三角函数情形统一规律
y*(x) = ea₀x · xk · (Q¹ₙcos b₀x + Q²ₙsin b₀x)

k = λ₀ = a₀+ib₀ 作为特征根的重数（一般方程中 k=0 或 1）。

3.4 叠加原理

线性叠加性

若 f(x) = f₁(x) + f₂(x)，则分别求出对应 f₁ 的特解 y₃ 和对应 f₂ 的特解 y₄，叠加得到特解：

y* = y₃(x) + y₄(x)
例  |  y″ + y = sin 3x + 2cos x  （两个右端项的叠加）

特征方程 λ²+1=0，λ₁,₂ = ±i。

对 sin 3x：λ₀ = 3i 不是特征根（情形4），设 y₃ = a cos 3x + b sin 3x。

代入求得 a=0，b=−1/8，故 y₃ = −(1/8)sin 3x。

对 2cos x：λ₀ = i 是特征根（情形5），设 y₄ = x(c cos x + d sin x)。

代入求得 c=0，d=1，故 y₄ = x sin x。

合并设特解：y* = a cos 3x + b sin 3x + x(c cos x + d sin x)

y = C₁cos x + C₂sin x − (1/8)sin 3x + x sin x

3.5 特解模板总结

f(x) 的形式λ₀ 是否为特征根特解模板 y*
eλ₀xPₙ(x) 不是特征根（k=0） eλ₀xQₙ(x)
单实根（k=1，Δ>0） eλ₀x·x·Qₙ(x)
二重实根（k=2，Δ=0） eλ₀x·x²·Qₙ(x)
ea₀x(P¹ₙcos b₀x + P²ₙsin b₀x) λ₀=a₀+ib₀ 不是复特征根（k=0） ea₀x(Q¹ₙcos b₀x + Q²ₙsin b₀x)
λ₀=a₀+ib₀ 是复特征根（k=1） ea₀x·x·(Q¹ₙcos b₀x + Q²ₙsin b₀x)
SECTION 04

n 阶常系数方程推广

4.1 n 阶齐次方程的特征根法

y(n) + a₁y(n−1) + ⋯ + aₙ₋₁y′ + aₙy = 0     (**)

对应的 n 次特征方程：

λⁿ + a₁λⁿ⁻¹ + ⋯ + aₙ₋₁λ + aₙ = 0
特征根与独立解的对应规则

实特征根

若 λ = λ₁ 是 k 重实特征根，则对应 k 个独立解：

eλ₁x,  xeλ₁x,  x²eλ₁x,  …,  xk−1eλ₁x

复特征根

若 λ = a±bi 是 k 重共轭复根，则对应 2k 个独立解：

eaxcos bx， xeaxcos bx， …， xk−1eaxcos bx
eaxsin bx， xeaxsin bx， …， xk−1eaxsin bx

关键性质：不同特征值对应的解一定线性独立，所有特征根提供的独立解总数恰好等于方程的阶数 n。

4.2 例题精讲

例1  |  y(4) − 2y‴ + 2y′ − y = 0  （4阶）

特征方程：λ⁴ − 2λ³ + 2λ − 1 = 0

分解：注意到 λ=1 是根，除法得 (λ−1)(λ³−λ²−λ+1) = (λ−1)²(λ²−1)·… 实际上：

λ⁴−2λ³+2λ−1 = (λ−1)³(λ+1)，得 λ₁=1（三重），λ₂=−1（单根）

λ=1（三重）→ 3个解：ex，xex，x²ex

λ=−1（单根）→ 1个解：e−x

y = (C₁ + C₂x + C₃x²)ex + C₄e−x
例2  |  y(5) − y(4) + 2y‴ − 2y″ + y′ − y = 0  （5阶）

特征方程：λ⁵ − λ⁴ + 2λ³ − 2λ² + λ − 1 = 0

分解：尝试 λ=1 代入 = 1−1+2−2+1−1=0 ✓，提取 (λ−1)：

= (λ−1)(λ⁴+2λ²+1) = (λ−1)(λ²+1)²

得 λ₁=1（单根），λ₂,₃=±i（二重复根）

λ=1 → ex（1个）

λ=±i 二重 → cos x，x cos x，sin x，x sin x（2×2=4个）

y = (C₁+C₂x)cos x + (C₃+C₄x)sin x + C₅ex

4.3 n 阶非齐次方程的特解

规则（与二阶完全一致，只是 k 的取值范围可能更大）

f(x) = eλ₀xPₘ(x)：

y* = eλ₀x·xk·Qₘ(x)

k = λ₀ 作为特征根的重数

f(x) = ea₀x(P¹ₘcos b₀x + P²ₘsin b₀x)：

y* = ea₀x·xk·(Q¹ₘcos b₀x+Q²ₘsin b₀x)

k = λ₀=a₀+ib₀ 作为特征根的重数

例3  |  y(6) + y(5) − y(4) = x−1  （λ₀=0，Pₘ=x−1 为1次）

特征方程：λ⁶+λ⁵−λ⁴=0 → λ⁴(λ²+λ−1)=0

λ=0（四重根），λ=(−1±√5)/2（另外两个单实根）

λ₀=0 是四重特征根（k=4），设特解：

y* = x⁴(ax+b)（x⁴ · 一次多项式）

展开代入，化简得：120a − (240ax+48b) = x−1（高阶导数项消掉了）

比较：x¹：−240a=1 → a=−1/240；x⁰：120a−48b=−1 → b=1/96

SECTION 05

可降阶的二阶变系数线性方程

对于一般的变系数二阶方程，没有通用解法，但存在两种可降阶的特殊结构。

降阶法 I：不含 y(x) 的项

方程形式
y″(x) + p(x)y′(x) = f(x)    （方程中无 y(x) 项）
  1. 设 u(x) = y′(x)，则 u′(x) = y″(x)
  2. 原方程化为：u′(x) + p(x)u(x) = f(x)，这是关于 u 的一阶线性方程（用积分因子法解）
  3. 解出 u(x)（含常数 C₁）
  4. 再积分：y(x) = ∫u(x)dx + C₂（加第二个常数）

降阶法 II：已知齐次方程一个特解

方程形式
y″(x) + p(x)y′(x) + q(x)y(x) = f(x)    (****)

前提条件：已知对应齐次方程有一个非零特解 y*(x)（即 y*满足 y*″+py*′+qy*=0）。

  1. 设非齐次方程的解为 y(x) = y*(x)·u(x)（用已知特解乘以待定函数）
  2. 代入方程 (****)，计算 y′, y″ 并展开：
    y′ = y*′u + y*u′，  y″ = y*″u + 2y*′u′ + y*u″
    代入后，含 u 的项（因为 y* 满足齐次方程）自动抵消
  3. 化简得：u″y* + (2y*′ + p(x)y*)u′ = f(x)
    这正好是降阶法 I 的形式（关于 u′ 的一阶线性方程）！
  4. 设 v = u′，按降阶法 I 解出 v，再积分两次得 y
适用场景

变系数方程通常靠观察或物理/几何背景猜出一个齐次特解（例如已知 y*=x 是某方程的解），然后用这个方法找全部解。是变系数方程中少数可以系统求解的情形之一。

高频易错点

❌ 错误1：忘检查 λ₀ 是否是特征根

设特解前必须先解特征方程！λ₀ 是否为特征根（及重数）直接决定特解模板中 x 的幂次。

❌ 错误2：重根通解写错

Δ=0 时通解是 (C₁+C₂x)eλx，不是 C₁eλx+C₂eλx（后者只有一个独立常数）。

❌ 错误3：三角函数特解少设一个

f(x) 只含 cos 时，特解中仍需同时设 cos 和 sin，因为求导会产生另一个。

❌ 错误4：多项式次数设低了

Qₙ 的次数必须和 Pₙ（f(x) 中多项式）相同，不能少设。

❌ 错误5：n阶方程特根重数判断错

n 阶方程中，若 λ₀ 是 k 重特征根，特解要乘 xk，k 可能是 0/1/2/3…

❌ 错误6：代入比较系数时计算错

设好特解后，求导代入展开是最易出错的步骤，务必耐心逐项展开、合并同类项再比较。

CHAPTER 01

基本概念与术语

什么是微分方程

含有未知函数 y = y(x) 及其导数的方程称为微分方程。本章只讨论常微分方程（ODE）——未知函数只含一个自变量。

ODE — 本章考点

未知函数只有一个自变量，如 y = y(x) 或 P = P(t)

例：y'' + y = sin x

PDE — 本章不考

未知函数有多个自变量，导数为偏导数。

例：zx + zy = 0

属性面板：读懂一道题的标签

阶数（Order）

方程中出现的最高阶导数的阶数。

一阶
y' − 2x + 1 = 0
二阶（高阶）
y'' + y = x

线性 vs 非线性

y 及各阶导数都是一次方，且不相互相乘 → 线性。否则 → 非线性。

✓ 线性
✗ 非线性
y' + p(x)y = f(x)
出现 y²、yy'、sin y

齐次 vs 非齐次（仅对线性方程）

齐次（f(x) = 0）
非齐次（f(x) ≠ 0）
无外力驱动的系统
有外力驱动的系统

解的分类

通解 — General Solution

含有独立任意常数的解。几阶方程的通解，就必须含几个独立的常数。

⚠ 常数必须独立

错  y = C₁ + C₂（本质只是一个常数）

对  y = C₁x + C₂（两者真正独立）

特解 — Particular Solution

利用附加条件确定了所有常数，不含任意常数。

初值条件：条件在同一点给出，如 y(0)=0, y'(0)=1

边界条件：条件在不同点给出，如 y(0)=0, y(1)=1

全章最重要的做题习惯

🔴 铁律：每次积分，立刻加 +C

不能等到最后再补。因为 C 后续可能参与指数运算（如 eC），晚加一步全盘皆输。

❌ 错误习惯（不定积分时）
✅ 正确习惯（解 ODE 时）
y = ∫(2x−1)dx = x²−x
最后再加 C
y = ∫(2x−1)dx + C = x²−x+C
积分号消失那一刻就加
CHAPTER 02

变量可分离方程

识别特征

方程能化为 y' = f(x)·g(y) 的形式，即 x 的函数与 y 的函数可以"各找各妈"，完全分开到等号两侧。

识别 y'=f(x)g(y)
→
令 g(y)=0 查特殊解
→
分离 dy/g(y)=f(x)dx
→
两边积分（立刻+C）

完整解题步骤

  1. 防遗漏检查：令 g(y) = 0，解出常数解 y = c，验证是否满足原方程。这类解有时无法被通解包含，必须单独列出。
  2. 假设 g(y) ≠ 0，分离变量：将 y' 写成 dy/dx，移项得 dy/g(y) = f(x)dx
  3. 两边同时积分：∫ dy/g(y) = ∫ f(x)dx + C（立刻加 C！）
  4. 化简：若积分出现 ln|y|，直接写成 y = Ce…，把正负号吸收进 C，无需讨论绝对值。

例题精讲

例 1 — 基础型  |  y' + xy² = 0

整理：dy/dx = −xy²，右边 = (−x)·(y²)，可分离。

Step 1 防遗漏：y² = 0 → y = 0，代入 0 + x·0 = 0 ✓，y=0 是解，记在旁边。

Step 2 分离（y≠0）：dy/y² = −x dx

Step 3 积分：−1/y = −x²/2 + C₁（立刻加 C₁）

Step 4 整理：y = 2/(x² + C)（C = −2C₁，重命名常数）

通解：y = 2/(x²+C)  及  y = 0（y=0 不被通解包含，需单独列）
例 2 — 含对数型  |  y' + 3y = 0

dy/y = −3 dx

ln|y| = −3x + C₁

y = eC₁·e−3x

把 ±eC₁ 合并为新常数 C（C 可正可负可为任意实数）。

y = Ce−3x
例 3 — 带初值  |  xy dx + (x²+1)dy = 0,  y(0)=1

整理：dy/y = −x dx/(x²+1)

积分：左边 ln|y|，右边 −½ln(x²+1)

ln y = −½ln(x²+1) + ln C  →  y = C/√(x²+1)

代初值：y(0) = C/1 = 1 → C = 1

特解：y = 1/√(x²+1)
积分技巧

把积分常数写成 ln C 形式（而非直接写 C），可以方便和左边的 ln 合并，省去大量化简步骤。

CHAPTER 03

一阶线性微分方程

标准形式（识别特征）
y' + p(x)y = f(x)
  • y' 的系数必须为 1（不是则先除过去）
  • 等号右边只有 x 的函数，没有 y
  • y 和 y' 都只出现一次方

三种方法对比

原理型方法（理解原理用）

  1. 解齐次方程 y' + p(x)y = 0，得通解 Y = Ce−∫p(x)dx
  2. 把常数 C 换成未知函数 u(x)，设 y = u(x)·e−∫p(x)dx
  3. 代入原方程，经一顿抵消，得 u'(x) = f(x)·e∫p(x)dx
  4. 积分求 u(x)，代回得 y
缺点

繁琐，每次都要从头推，不推荐在考试中使用。

死记硬背型（不推荐）

y = e−∫p(x)dx · (∫ f(x)·e∫p(x)dx dx + C)
缺点

时间长了容易记错，积分因子法推导更安全，建议用方法3。

⭐ 强烈推荐：积分因子法

  1. 求积分因子：μ(x) = e∫p(x)dx（此处积分不加 C）
  2. 两边同乘 μ(x)：左边自动凑成完美导数 → (y·μ(x))' = f(x)·μ(x)
  3. 两边积分：y·μ(x) = ∫ f(x)·μ(x) dx + C（这里加 C）
  4. 除以 μ(x) 得最终的 y
为什么乘以 μ 后左边变成 (yμ)'？

对 (y·μ)' 展开：y'μ + y·μ' = y'μ + y·p(x)μ = μ(y' + p(x)y)

这正好是原方程左边乘以 μ！配凑是链式法则的必然结果，不是凑巧。

例题精讲（积分因子法）

例  |  y' − 2y/(x+1) = (x+1)³

Step 1 识别：p(x) = −2/(x+1)，f(x) = (x+1)³

Step 2 求 μ：

μ(x) = e∫(−2/(x+1))dx = e−2ln(x+1) = (x+1)−2

Step 3 乘以 μ，左边直接写成导数：

( y · 1/(x+1)² )' = (x+1)³ · 1/(x+1)² = (x+1)

Step 4 两边积分（左边去撇，右边积分并立刻+C）：

y/(x+1)² = ½(x+1)² + C

Step 5 化简（两边乘以 (x+1)²）：

y = ½(x+1)⁴ + C(x+1)²

神级技巧：x 与 y 互换

⚠ 当题目"什么类型都不像"时

遇到如 y' = y/(y³+x) 这类方程，既不可分离，也不是 y 的线性方程，尝试把 x 当未知函数，y 当自变量。

  1. 设 x = x(y)，利用 y'(x) = 1/x'(y)
  2. 代入原方程得：x' = (y³+x)/y = y² + (1/y)x
  3. 移项：x' − (1/y)x = y² ← 这正是关于 x 的一阶线性方程！
  4. 用积分因子法秒杀
CHAPTER 04

变量代换（三种换元法）

有些方程"乔装打扮"，换元后才能变成熟悉的可分离或线性方程。

方程特征 换元类型 令什么变量 化为
右边能凑成 f(y/x) 齐次方程 u = y/x 变量可分离
y' + py = qya（a≠0,1） Bernoulli 方程 u = y1−a 一阶线性
x,y 绑在一起，如 cos(x+y) 整体换元（观察法） 令整体 = u 变量可分离

类型 1：齐次方程

识别特征

y' = f(y/x)。所有 x, y 的组合次数相同，能凑成 y/x 整体，等号右边没有单独游离的 x 或 y。

快速判断法：将 x 换成 tx，y 换成 ty，若方程不变（t 消掉），则是齐次方程。

  1. 令 u = y/x，即 y = ux
  2. 对 x 求导（链式法则）：y' = u'x + u
  3. 代入原方程：u'x + u = f(u)
  4. 整理：du/(f(u)−u) = dx/x → 变量可分离！
  5. 积分，解出 u，换回 y = ux
例  |  y' = (x+y)/x

整理：y' = 1 + y/x，令 u = y/x

u'x + u = 1+u → u'x = 1 → du = dx/x

积分：u = ln|x| + C

y = x(ln|x| + C)

类型 2：Bernoulli（贝努利）方程

识别特征
y' + p(x)y = q(x)·ya    （a ≠ 0, 1）

比一阶线性方程多了一个 ya 的尾巴。a=0 退化成线性；a=1 退化成可分离，均单独处理。

  1. 若 a > 0，先记录 y ≡ 0 是一个解
  2. 两边除以 ya：y−ay' + p(x)y1−a = q(x)
  3. 令 u = y1−a，则 u' = (1−a)y−ay'
  4. 代入，得关于 u 的一阶线性方程
  5. 用积分因子法解出 u，换回 y = u1/(1−a)
⚠ 常见错误

换元后，不要忘了 y−ay' 整体被替换为 u'/(1−a)，整理时系数是 1/(1−a)，不是 1。

类型 3：整体换元（观察法）

识别特征

x 和 y 被"绑在一起"出现，如 y' = cos(x+y) 或 y' = (x+y)²。

  1. 观察"绑在一起"的整体，令它等于 u（如令 u = x+y）
  2. 对 x 求导，用 u' 替换 y'（如 u' = 1 + y' → y' = u'−1）
  3. 原方程化为关于 u 的可分离方程
例  |  y' = cos(x+y)

令 u = x+y → u' = 1+y' → y' = u'−1

代入：u'−1 = cos u → u' = 1+cos u

利用 1+cos u = 2cos²(u/2)：

du/(2cos²(u/2)) = dx → ½tan(u/2) = x + C

tan((x+y)/2) = 2x + C
CHAPTER 05

可降阶的高阶方程

标准二阶方程含有 x, y, y', y''。只要它"缺项"，就可以降阶处理。

缺什么 标准形式 代换 注意
缺 y 和 y' y'' = f(x) 直接积分两次 每次积分都加新常数
缺 y（不显含 y） y'' = f(x, y') p = y'，p' = y''（p 看作 x 的函数） 化为关于 p(x) 的一阶方程
缺 x（不显含 x） y'' = f(y, y') p = y'，y'' = p·dp/dy（p 看作 y 的函数） 高频易错：链式法则！

类型 1：缺 y 和 y'（y'' = f(x)）

最简单：直接积分两次

y' = ∫f(x)dx + C₁（第一次积分，立刻加 C₁）

y = ∫(∫f(x)dx + C₁)dx + C₂ = … + C₁x + C₂（第二次积分，加 C₂）

类型 2：缺 y（y'' = f(x, y')）

  1. 令 p = y'，则 y'' = p'（= dp/dx），p 看作 x 的函数
  2. 原方程变为 p' = f(x, p)，是关于 p(x) 的一阶方程
  3. 解出 p(x, C₁)
  4. 由 y' = p，再积分一次得 y（加 C₂）
例  |  y'' = 2xy'（右边只有 x 和 y'，无 y）

令 p = y'，则 p' = 2xp（可分离方程）

dp/p = 2x dx → ln|p| = x² + C₁ → p = C₁ex²

（合并常数 ±eC₁ 为 C₁）

y = C₁∫ex²dx + C₂

类型 3：缺 x（y'' = f(y, y')）⭐ 最难最易错

🔴 超级易错点：两种缺项的 y'' 替换方式完全不同！
缺 y 时（p 是 x 的函数）
缺 x 时（p 是 y 的函数）
y'' = dp/dx = p'
y'' = p·dp/dy（链式法则！）

推导：y'' = dp/dx = (dp/dy)·(dy/dx) = (dp/dy)·p = p·dp/dy

记忆口诀：缺 x → p·dp/dy；缺 y → dp/dx = p'。两者截然不同。

  1. 令 p = y'，把 p 看作 y 的函数 p = p(y)
  2. 代换：y'' = p·dp/dy
  3. 原方程化为 p·dp/dy = f(y, p)，是关于 p, y 的一阶方程
  4. 解出 p(y, C₁)
  5. 由 dy/dx = p(y)，分离变量 dy/p(y) = dx，再积分（加 C₂）
经典例题  |  yy'' − (y')² = 0

判断：含 y, y', y''，无独立 x → 缺 x 型。

代换：令 p = y', y'' = p·dp/dy，代入：

y·p·(dp/dy) − p² = 0  →  p·(y·dp/dy − p) = 0

路线 A（p=0）：y' = 0 → y = C（常数解）

路线 B（y·dp/dy = p）：

dp/p = dy/y → ln|p| = ln|y| + C₁ → p = C₁y

换回：y' = C₁y → dy/y = C₁dx → y = C₂eC₁x

通解：y = C₂eC₁x（令 C₁=0 可包含路线 A 的常数解）
CHAPTER 06

应用与建模

应用题核心是把中文翻译成微分方程，列方程后，回归前面章节的解法。

几何应用

⚠ 最容易犯的错误

题目说"曲线上任意一点"，必须用 (x₀, y₀) 表示，把 x, y 留给切线方程！

切线方程：y − y₀ = y'(x₀)·(x − x₀)

列出几何条件（切线截距、曲线长度等），用 x₀, y₀ 建立等式，化简后去掉下标 0，得微分方程。

人口演化模型

模型一：Malthus 模型

假设

出生率 b，死亡率 h 均为常数，净增长率 r = b − h。

方程推导

[P(t+Δt)−P(t)]/Δt ≈ rP(t)

Δt→0 即得：

P'(t) = rP(t)
求解（分离变量）
P(t) = P₀·ert
致命缺陷

r > 0 时人口无限指数爆炸，不符合资源有限的现实。

模型二：Logistic 模型

改进

引入最大承载力 Pmax，增长率随人口增加而线性递减：

ractual = r(1 − P/Pmax)

方程
P' = r(1 − P/Pmax)·P
最终公式
P(t) = P₀·Pmax / (P₀ + (Pmax−P₀)e−rt)

当 t→∞，P(t)→Pmax。增长曲线呈 "S" 型。

Logistic 模型：部分分式积分详解

关键步骤：裂项积分

分离变量后，左边被积函数为 1/[P(1−P/Pmax)]，化简后：

Pmax/[P(Pmax−P)] = 1/P + 1/(Pmax−P)

验证（通分）：[(Pmax−P)+P]/[P(Pmax−P)] = Pmax/[P(Pmax−P)] ✓

分别积分：ln P − ln(Pmax−P) = rt + C

即 ln[P/(Pmax−P)] = rt + C，两边取指数，代入初值 P(0)=P₀，解出 P(t)。

高频易错点汇总

错误 1：忘记加 +C

积分完成那一刻立刻加 C，不能等到最后。

错误 2：漏掉特殊解

令 g(y)=0 的常数解，有时无法被通解包含，须单独列出。

错误 3：缺 x 时 y'' 替换错

缺 x 时 y''=p·dp/dy；缺 y 时 y''=dp/dx。两者截然不同。

错误 4：通解常数不独立

二阶方程通解必须含两个独立常数。C₁+C₂ 不独立，C₁x+C₂ 才独立。

错误 5：几何题切点写 (x,y)

切点必须用 (x₀,y₀)，把 x,y 留给切线方程，否则混乱。

错误 6：积分因子的积分加了 C

求 μ = e∫p(x)dx 时，指数上的积分不加 C。C 只在最后方程积分时加。

常用积分结果速查

三角与反三角

∫1/(1+u²)du = arctan u + C

∫1/cos²u du = tan u + C

∫1/√(1−u²)du = arcsin u + C

1+cos u = 2cos²(u/2)

对数与指数

∫1/u du = ln|u| + C

∫un du = un+1/(n+1) + C  (n≠−1)

∫eau du = eau/a + C

∫1/(a²+u²)du = (1/a)arctan(u/a) + C

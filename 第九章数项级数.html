SECTION 01

收敛定义与基本性质

1.1 级数的本质

芝诺悖论（阿基里斯追乌龟）提示我们：无穷多个有限量相加，结果可以是一个有限常数。数项级数用严谨的极限语言来回答"无穷多个数加起来等于多少"。

基本定义

给定数列 {aₙ}，定义部分和数列 Sₙ = a₁ + a₂ + ⋯ + aₙ。

若 limn→∞ Sₙ = S（有限数存在），称 ∑aₙ 收敛，S 是级数的和。

若极限不存在或为 ±∞，称 ∑aₙ 发散。

核心公理（贯穿全章）

级数的收敛性与前面有限项完全无关。增删改变有限项只影响收敛时的"和"，不影响"是否收敛"。

对正项级数：Sₙ 单调递增，故收敛 ⟺ Sₙ 有上界；要么收敛，要么发散到 +∞。

1.2 两个基本模型

等比级数（几何级数）

部分和 Sₙ = (1−qⁿ)/(1−q)

|q| < 1：收敛，和 = 1/(1−q)

|q| ≥ 1：发散

p-级数（最重要的参照标尺）
∑ 1/nᵖ

p > 1 收敛  |  p ≤ 1 发散

p = 1 即调和级数，发散！

1.3 收敛的必要条件

⚠ 必要条件 — 逆否才是做题武器

定理：∑aₙ 收敛 ⟹ lim aₙ = 0

逆否（做题用）：lim aₙ ≠ 0 ⟹ ∑aₙ 必然发散

陷阱 反之不成立！aₙ → 0 只是必要条件，不充分。最经典反例：调和级数 ∑1/n，通项趋0但发散。

调和级数发散的严格证明（Cauchy准则法）

取 n = 2m，考察第 m+1 项到第 2m 项的和：

1/(m+1) + 1/(m+2) + ⋯ + 1/(2m) ≥ m · 1/(2m) = 1/2

无论 m 多大，长度为 m 的一段的和始终 ≥ 1/2，不满足 Cauchy 收敛准则。

这个证明思路务必掌握——是 Cauchy 准则"证发散"的典型用法。

1.4 Cauchy 收敛准则（充要条件）

Cauchy 收敛准则

∑aₙ 收敛 ⟺ 对任意 ε > 0，存在 N，当 n > m > N 时：

|aₘ₊₁ + aₘ₊₂ + ⋯ + aₙ| < ε

即：从任意远处开始，任意有限段的和都可以任意小。

1.5 级数的基本性质

性质内容注意
线性性∑aₙ=A, ∑bₙ=B 均收敛 ⟹ ∑(kaₙ+lbₙ) = kA+lB两收敛级数可线性组合
加括号收敛级数加括号后仍收敛，和不变逆命题不成立
去有限项增删有限项，敛散性不变和会改变
SECTION 02

上下极限（limsup 与 liminf）

2.0 为什么需要上下极限

许多数列（如 aₙ = cos(2nπ/5)）根本没有普通极限，但它们的"振荡范围"是有规律的。普通极限要求所有子列都收敛到同一个值，而上下极限只盯住"最大的"和"最小的"那两个聚集方向——它们对任意数列都存在（可以取 ±∞），是比普通极限更基本的概念。

本节路线图

极限点（子列收敛的目标）→ 极限点集 E → sup E, inf E 也在 E 里 → 定义上/下极限 = max E / min E → 特征刻画 → 性质与运算 → 广义推广（去掉有界性）

2.1 极限点与极限点集

定义 · 极限点（凝聚点）

称 ξ ∈ ℝ 为数列 {aₙ} 的一个极限点，若存在 {aₙ} 的子列 {aₙₖ}，使得

aₙₖ → ξ，（k → ∞）

记 {aₙ} 的极限点集为 E = { ξ | ξ 为 {aₙ} 的极限点 }。

极限点也称凝聚点。注意：这里 ξ ∈ ℝ（有限实数），不含 ±∞；后面广义版本才允许 ±∞。

等价刻画定理

ξ ∈ E   ⟺   对任意 δ > 0，集合 S(δ) = { n | aₙ ∈ O(ξ, δ) } 是无限集。

（O(ξ, δ) = (ξ−δ, ξ+δ) 表示 ξ 的 δ-邻域）

换句话说：ξ 是极限点 ⟺ 无论多小的邻域，总有无限项落进去。

证明思路（"⇐"方向的子列构造——关键！）

已知：对每个 δ > 0，S(δ) 均为无限集。
利用 δ 的任意性，递归地构造严格递增下标的子列（注意保证 nₖ 严格递增，这是关键！）：

  • 任取 n₁ ∈ S(1)；
  • 任取 n₂ ∈ S(1/2) ∩ { n > n₁ }；  （S(1/2) 是无限集，故 { n > n₁ } 内有元素）
  • 任取 n₃ ∈ S(1/3) ∩ { n > n₂ }；
  • 一般地，任取 nₖ ∈ S(1/k) ∩ { n > nₖ₋₁ }。

则 |aₙₖ − ξ| < 1/k → 0，（k → ∞），故 aₙₖ → ξ，即 ξ ∈ E。  ∎

三个典型例子

(1) aₙ = (−1)ⁿ⁻¹：奇数项 = 1，偶数项 = −1，故极限点集 E = {1, −1}。

(2) {aₙ} 是有理数集 ℚ 的一个排列：ℝ 中每一点都有 ℚ 中的序列逼近，故 E = ℝ。

(3) aₙ = n：趋于 +∞，没有任何有限收敛子列，故 E = ∅（有界性不满足）。

2.2 上下极限的定义（有界数列版）

定理：有界数列的极限点集非空，且 sup E、inf E 都属于 E

设 {aₙ} 有界，则极限点集 E 非空有界，且

sup E ∈ E，  inf E ∈ E

即 E 的上确界和下确界都能被子列真正"达到"，故 sup E = max E，inf E = min E。

证明（以 sup E 为例）

记 β = sup E。对任意 δ > 0，由上确界定义，存在 ξ ∈ E 使 β − δ < ξ ≤ β。

取 δ' = ξ − (β − δ) > 0，则邻域 O(ξ, δ') ⊂ O(β, δ)。

又因 ξ ∈ E，O(ξ, δ') 含 {aₙ} 无限项，从而 O(β, δ) 也含 {aₙ} 无限项。

由等价刻画定理，β ∈ E。同理可证 inf E ∈ E。   ∎

定义 · 上极限与下极限

设 {aₙ} 有界，定义：

lim supn→∞ aₙ = limn→∞ aₙ  := max E  （最大极限点） lim infn→∞ aₙ = limn→∞ aₙ  := min E  （最小极限点）

记号说明：上极限常写作 limsup 或 lim，下极限常写作 liminf 或 lim。

2.3 上下极限的特征刻画

上极限 β = max E 有一个非常实用的等价刻画，是做题时的常用工具：

定理 · β 是上极限的充要条件

设 {aₙ} 有界，β ∈ ℝ，则

β = lim sup aₙ   ⟺   对任意 ε > 0：
① O(β, ε) 含 {aₙ} 无限项（β 处有无穷多项聚集）
② (β + ε, +∞) 至多含有限项（β 以上"几乎没有"项）

类似地，α = lim inf aₙ ⟺ ① O(α, ε) 含无限项；② (−∞, α − ε) 至多含有限项。

证明（"⇒" 方向：证②）

假设 (β + ε, +∞) 含 {aₙ} 无限项，由 Bolzano-Weierstrass 定理，存在子列 aₙₖ → ξ ≥ β + ε，则 ξ ∈ E 且 ξ > β = max E，矛盾。故②成立。

直觉理解

β = lim sup aₙ 意味着：β 是数列"最终到不了的最高处"。
· 数列无限次接近 β（条件①），但
· 超过 β 的地方，只有有限项（条件②）。

换句话说：β 是数列所有极限点里最大的那个，也是数列"几乎最终不超过"的门槛。

2.4 极限存在的充要条件

定理 · 极限与上下极限的关系

设 {aₙ} 有界，则

limn→∞ aₙ 存在   ⟺   lim sup aₙ = lim inf aₙ

此时极限值等于该公共值。

证明（"⇐"方向，用分离性）

设 lim sup aₙ = lim inf aₙ = A。对任意 ε > 0，由上下极限的特征刻画：

· (A + ε, +∞) 至多含有限项，所以从某 N₁ 起：aₙ < A + ε

· (−∞, A − ε) 至多含有限项，所以从某 N₂ 起：aₙ > A − ε

取 N = max{N₁, N₂}，则 n > N 时 |aₙ − A| < ε，即 lim aₙ = A。   ∎

重要推论：上下极限替代极限证明

从此以后，证明极限存在并求值，完全可以用计算上极限和下极限是否相等来替代。这往往比用 ε-N 语言直接证明更为方便——特别是在不等式估计中，取上/下极限是一种强大的操作。

2.5 保序性与分离性

保序性定理

设 {aₙ}、{bₙ} 有界，且从某 N₀ 起恒有 aₙ ≤ bₙ，则：

lim inf aₙ ≤ lim inf bₙ，  lim sup aₙ ≤ lim sup bₙ
分离性定理（所有判别法证明的根基）

设 {aₙ}、{bₙ} 有界，若 lim sup aₙ < lim inf bₙ，则存在 N₀，当 n > N₀ 时恒有：

aₙ < bₙ

证明：取 ε = (lim inf bₙ − lim sup aₙ)/2 > 0，则存在 N₀，当 n > N₀ 时：

aₙ < lim sup aₙ + ε = lim inf bₙ − ε < bₙ。   ∎

应用：比值法中，若 lim sup (aₙ₊₁/aₙ) = ρ < 1，则在 ρ 和 1 之间取 r，由分离性从某项起 aₙ₊₁/aₙ < r，从而 aₙ < a₁·rⁿ，再与等比级数比较得收敛。

注：实际应用中，{bₙ} 经常取成常数数列。

2.6 上下极限的运算性质

加法运算（一般不等式）

设 {aₙ}、{bₙ} 有界，则：

lim inf aₙ + lim inf bₙ   ≤   lim inf (aₙ + bₙ)   ≤   lim sup (aₙ + bₙ)   ≤   lim sup aₙ + lim sup bₙ

不等号一般不能取等！例：aₙ = (−1)ⁿ，bₙ = (−1)ⁿ⁺¹ = −aₙ，则 aₙ + bₙ = 0，但 lim sup aₙ + lim sup bₙ = 1+1 = 2。

加法运算（等号成立：一个极限存在时）

若 limn→∞ aₙ 存在（记为 A），则：

lim sup (aₙ + bₙ) = A + lim sup bₙ
lim inf (aₙ + bₙ) = A + lim inf bₙ

证明思路（下极限）：一方面由一般不等式有 lim inf(aₙ+bₙ) ≥ A + lim inf bₙ。另一方面，lim inf bₙ = lim inf((aₙ+bₙ)−aₙ) ≥ lim inf(aₙ+bₙ) − A，故 lim inf(aₙ+bₙ) ≤ A + lim inf bₙ。两方向合并得等式。 ∎

乘法运算（aₙ, bₙ ≥ 0 时）

设 aₙ, bₙ ≥ 0 有界，则：

lim inf aₙ · lim inf bₙ   ≤   lim inf (aₙbₙ)   ≤   lim sup (aₙbₙ)   ≤   lim sup aₙ · lim sup bₙ

同样，若 lim aₙ 存在（= A ≠ 0），则 lim sup (aₙbₙ) = A · lim sup bₙ，等号成立。

2.7 广义极限点与广义上下极限

前面的定义要求数列有界，但我们希望对任意数列都能定义上下极限。

定义 · 广义极限点集 Ẽ

将极限点的定义扩展到 ξ ∈ ℝ̃ = ℝ ∪ {+∞, −∞}：称 ξ ∈ ℝ̃ 为 {aₙ} 的广义极限点，若存在子列 aₙₖ → ξ（允许趋于 ±∞）。

记广义极限点集为 Ẽ，显然 E ⊂ Ẽ。

规定扩展运算：对任意 x ∈ ℝ 有 −∞ < x < +∞；若 A ⊂ ℝ̃ 含 +∞ 或 A∩ℝ 无上界，则 sup A = +∞；若含 −∞ 或无下界，则 inf A = −∞。

定理：Ẽ 包含自身的上/下确界
sup Ẽ ∈ Ẽ，  inf Ẽ ∈ Ẽ

关键情形：当 Ẽ∩ℝ 非空无上界时，+∞ ∈ Ẽ（即存在子列趋于 +∞）。这可以用"取 n₁ 使 aₙ₁ > 1，取 n₂ > n₁ 使 aₙ₂ > 2，……"的构造来严格证明。

重新定义（无需有界性）· 上下极限

对任意实数列 {aₙ}（无需有界），定义：

lim supn→∞ aₙ = max Ẽ  （允许取 +∞） lim infn→∞ aₙ = min Ẽ  （允许取 −∞）

有了这个定义，任何数列都有上下极限（可以是 ±∞），前面关于有界数列的所有定理仍然成立（运算中需避免未定型如 (+∞)+(−∞)）。

广义极限存在的充要条件
limn→∞ aₙ = ξ₀ (∈ ℝ̃)   ⟺   lim sup aₙ = lim inf aₙ = ξ₀   ⟺   Ẽ = {ξ₀}
⚠ 易混淆：E = {ξ₀} 不等于极限等于 ξ₀！

必须是广义极限点集 Ẽ = {ξ₀} 才能推出极限存在。若仅有狭义极限点集 E = {ξ₀}，结论不成立。

反例：aₙ = n^(−1)ⁿ = 1, n, 1, n, 1, n,…（n 为偶数时 aₙ = n，n 为奇数时 aₙ = 1）

狭义极限点集 E = {1}（只有有限子列收敛），但 Ẽ = {1, +∞}，lim inf = 1，lim sup = +∞，极限不存在！

2.8 用上下极限证题的典型方法

例1（课件） |  求 xₙ = cos(2nπ/5) 的上下极限

cos(2nπ/5) 以 5 为周期，列出一个周期内的值：

n mod 512340(5)
xₙ cos(2π/5) cos(4π/5) = −cos(π/5) cos(6π/5) = −cos(π/5) cos(8π/5) = cos(2π/5) cos(2π) = 1

故极限点集 E = { cos(2π/5), −cos(π/5), 1 }，其中最大值为 1，最小值为 −cos(π/5)。

lim sup xₙ = 1，  lim inf xₙ = −cos(π/5)
例2（课件） |  证明 limn→∞ ∫₀^(π/2) sinⁿx dx = 0

记 Iₙ = ∫₀^(π/2) sinⁿx dx。对任意 ε ∈ (0, π/2)，将积分拆开估计：

0 ≤ Iₙ = ∫₀^(π/2−ε) sinⁿx dx + ∫_(π/2−ε)^(π/2) sinⁿx dx ≤ sinⁿ(π/2−ε) · π/2 + ε

固定 ε，因为 sin(π/2−ε) < 1，故 sinⁿ(π/2−ε) → 0（n→∞）。
对不等式两端同时取上下极限：

0 ≤ lim inf Iₙ ≤ lim sup Iₙ ≤ 0 · π/2 + ε = ε

令 ε → 0⁺，得 0 ≤ lim inf Iₙ ≤ lim sup Iₙ ≤ 0，故 lim inf = lim sup = 0。

limn→∞ ∫₀^(π/2) sinⁿx dx = 0  （上下极限相等，极限存在且为 0）
技巧：先用参数 ε 将"大头"（靠近 π/2 的部分）隔离，然后用上极限 ≤ ε 再令 ε→0，是经典的两步夹逼法。
例3（Cesàro均值） |  已知 lim aₙ = 0，证明 lim (a₁+⋯+aₙ)/n = 0

对任意 ε > 0，由 aₙ → 0，存在 N 使 n > N 时 |aₙ| < ε。
当 n > N 时：

|(a₁+⋯+aₙ)/n| ≤ |a₁+⋯+aₙ|/n ≤ |a₁+⋯+aₙ|/n ≤ |a₁+⋯+a_N|/n + ε

固定 ε（从而固定 N），对右端取上极限：前一项 |a₁+⋯+aₙ|/n → 0（N 固定，分母趋∞），故

0 ≤ lim sup |(a₁+⋯+aₙ)/n| ≤ 0 + ε = ε

令 ε → 0⁺ 得 lim sup = 0，而 lim inf ≥ 0，故极限存在且为 0。

lim (a₁+⋯+aₙ)/n = 0  （均值收敛到同一极限）
这个结论可以推广：若 lim aₙ = A（任意有限数），则 lim (a₁+⋯+aₙ)/n = A。用 bₙ = aₙ − A 代入即得。

2.9 关键引理（判别法的基础）

比值 ≤ 根值（根值法比比值法更强）
lim inf aₙ₊₁/aₙ   ≤   lim inf ⁿ√aₙ   ≤   lim sup ⁿ√aₙ   ≤   lim sup aₙ₊₁/aₙ

含义：比值法"判断不了"（lim sup = 1）的情形，根值法有时仍能判断；比值法能判断的，根值法一定也能判断。

实用建议：通项含 n! 优先用比值法（阶乘消得干净）；含 nⁿ 优先用根值法（ⁿ√(nⁿ) = n）。

总结：上下极限的本质

上极限 = 数列"无穷次到达"的最高点；下极限 = 数列"无穷次到达"的最低点。

它们是数列振荡的"天花板"和"地板"，普通极限存在当且仅当天花板等于地板。

上下极限的最大价值：在不知道极限是否存在时，仍能做不等式估计——这是它贯穿整个级数理论的原因。

SECTION 03

正项级数判别法（五大武器）

无穷大量的碾压关系（必背）
ln n ≪ nᵃ ≪ rⁿ ≪ n! ≪ nⁿ    （a > 0, r > 1）

含义：相对于 nᵃ，lnn 几乎等于 n⁰（常数级）；相对于 rⁿ，nᵃ 几乎等于 1ⁿ（被指数碾压）。这是所有等价替换和比较判别的直觉基础。

武器 1 比较判别法 + 等价无穷小 / Taylor展开 最常用

直接比较形式

0 ≤ aₙ ≤ bₙ（从某项起）：∑bₙ 收敛 ⟹ ∑aₙ 收敛；∑aₙ 发散 ⟹ ∑bₙ 发散。

课件原话的做法思路：欲证收敛，把 uₙ 放大一点变成简单的 vₙ，使 ∑vₙ 一眼看出收敛；欲证发散，把 uₙ 缩小一点变成简单的 vₙ，使 ∑vₙ 一眼看出发散。

极限比较形式（实战首选）

设 aₙ, bₙ > 0，λ = lim aₙ/bₙ：

0 < λ < +∞

同敛散

λ = 0

∑bₙ 收敛 ⟹ ∑aₙ 收敛

λ = +∞

∑bₙ 发散 ⟹ ∑aₙ 发散

常用等价无穷小速查（n→∞）

原式等价替换备注
sin(1/n), tan(1/n)∼ 1/n直接等价
1 − cos(1/n)∼ 1/(2n²)注意是 n² 量级！
ln(1+1/n)∼ 1/n（精确式：= 1/n − 1/(2n²)+…）两项相减必须展到 1/n²
e^(1/n) − 1∼ 1/n直接等价
(1+1/n)ᵅ − 1∼ α/n广义 Bernoulli
⚠ 课件重点易错：两等价量相减，必须多展一项 Taylor！

判断 ∑[1/n − ln(1+1/n)] 的敛散性：

错  用 ln(1+1/n) ∼ 1/n，通项 ∼ 0 → 无法判断。

对  展开到 1/n² 项：ln(1+1/n) = 1/n − 1/(2n²) + o(1/n²)

SECTION 04

任意项级数：绝对收敛与条件收敛

4.1 定义与逻辑关系

绝对收敛

∑|aₙ| 收敛 → 称 ∑aₙ 绝对收敛

定理：绝对收敛 ⟹ 收敛

证：0 ≤ |aₙ|−aₙ ≤ 2|aₙ|，比较知 ∑(|aₙ|−aₙ) 收敛，再用 aₙ=|aₙ|−(|aₙ|−aₙ) 即得。

条件收敛

∑aₙ 收敛，但 ∑|aₙ| 发散

深层本质：正项部分 ∑aₙ⁺ = +∞，负项部分 ∑aₙ⁻ = +∞。

条件收敛是正负两个无穷大精妙抵消的结果。

逻辑关系（考试高频）

绝对收敛 ⟹ 收敛（正确）  |  收敛 ⟹ 绝对收敛（错误，条件收敛是反例）

4.2 Leibniz 判别法（交错级数专用）

武器 Leibniz 判别法 正负交替型 ∑(−1)ⁿaₙ
Leibniz 级数 — 三个条件缺一不可
  1. aₙ > 0（通项正值）
  2. aₙ 单调递减（aₙ₊₁ ≤ aₙ）
  3. limn→∞ aₙ = 0

结论：Leibniz 级数必收敛，且 0 ≤ S ≤ a₁。

余项误差：|rₙ| ≤ aₙ₊₁，且 rₙ 与 (−1)ⁿaₙ₊₁ 同号。

证明思路（课件完整给出）

偶数项部分和 S₂ₙ：

S₂ₙ = (a₁−a₂)+(a₃−a₄)+⋯+(a₂ₙ₋₁−a₂ₙ) ≥ 0（单调减保证每括号 ≥ 0）

S₂ₙ = a₁−(a₂−a₃)−⋯−a₂ₙ ≤ a₁（有上界）

故 {S₂ₙ} 单调有界，收敛到 S。由 S₂ₙ₋₁=S₂ₙ−a₂ₙ → S，故 limSₙ=S，级数收敛。

例  |  ∑(−1)ⁿ⁻¹/nᵖ 的收敛性完整分析

p ≤ 0：1/nᵖ = n|ᵖ| → ∞，通项不趋于0，发散

p > 0：满足 Leibniz 三条件，收敛

进一步区分：p > 1 时 ∑1/nᵖ 收敛 → 绝对收敛；0 < p ≤ 1 时 ∑1/nᵖ 发散 → 条件收敛

⚠ 课件夺命易错题：变号级数不能用等价无穷小！

考察：(1) ∑(−1)ⁿ⁻¹/√n 与 (2) ∑[(−1)ⁿ⁻¹/√n + 1/n]

两者通项的比值趋于1，是等价的。但：

(1) 满足 Leibniz，收敛； (2) = Leibniz级数 + 调和级数，发散！

结论：正项级数可以用等价量替换；任意项级数绝对不可以！

SECTION 05

Abel 变换与 Dirichlet / Abel 判别法

5.1 Abel 变换（离散分部积分）

Abel 变换公式

设 Bₖ = b₁+b₂+⋯+bₖ，B₀=0，则：

∑k=1p aₖbₖ = aₚBₚ − ∑k=1p−1 Bₖ(aₖ₊₁−aₖ)

与积分分部 ∫u dv = uv − ∫v du 完全对应：aₖ → u，bₖ → dv，Bₖ → v。

Abel 引理（估计核心）

设 |Bₙ| ≤ M，{aₙ} 非负单调递减，则：

|∑k=1p aₖbₖ| ≤ M·a₁

这是 D/A 判别法的证明基础。实际应用中更常用的加强版：

|∑k=m+1n aₖbₖ| ≤ 2 · maxm<k≤n|Sₖ−Sₘ| · (|aₘ₊₁|+|aₙ|)

5.2 D-A 判别法（统一定理）

A-D 判别法（课件统一表述）

若下述两个条件之一满足，则 ∑aₙbₙ 收敛：

D 法 Dirichlet 判别法
条件

① {aₙ} 单调趋于 0（aₙ → 0）

② ∑bₙ 的部分和有界（|Bₙ| ≤ M）

口诀：对 aₙ 要求高（趋于0），对 bₙ 要求低（仅有界）。

A 法 Abel 判别法
条件

① {aₙ} 单调有界（不必趋于0）

② ∑bₙ 本身收敛

口诀：对 bₙ 要求高（已收敛），对 aₙ 要求低（仅单调有界）。

统一证明框架（课件完整给出）

由 Abel 引理加强版：|∑aₖbₖ| ≤ 2·max|Bₖ|·(|aₘ₊₁|+|aₙ|)

D 法：部分和有界 → max|Bₖ| ≤ M；aₙ→0 → |aₘ₊₁|+|aₙ| → 0 → 右端 → 0 → Cauchy 准则。

A 法：∑bₙ 收敛 → 尾段 Bₖ=Sₖ−Sₘ → 0；aₙ 有界 → 右端 → 0。

SECTION 07

Riemann 重排定理 · 级数乘法

7.1 Riemann 重排定理（震惊定理）

🔴 Riemann 重排定理

设 ∑xₙ 条件收敛，则对任意 a ≤ b（a,b ∈ ℝ∪{±∞}），存在 ∑xₙ 的一个更序 ∑xₙ'，使得：

limn→∞ S'ₙ 的下极限 = a， 上极限 = b

特别地：取 a=b，可以使更序级数收敛到任意实数；取 a=−∞,b=+∞，可以使其振荡发散。

证明构造（课件完整给出，理解思路即可）

由条件收敛知 ∑xₙ⁺=+∞，∑xₙ⁻=+∞，且 xₙ⁺,xₙ⁻→0。

以使级数收敛到给定 S（取 a=b=S）为例，构造如下：

  1. 取最少正项（x₁⁺,x₂⁺,…,xₙ₁⁺）使部分和 > S（因∑xₙ⁺=+∞，这样的 n₁ 存在）
  2. 取最少负项（x₁⁻,…,xₘ₁⁻）使部分和 < S（因∑xₙ⁻=+∞，这样的 m₁ 存在）
  3. 再取最少正项使超过 S，再取最少负项使低于 S……如此循环构造更序
  4. 因为 xₙ→0，每次"超调"越来越小（上下夹逼），部分和最终收敛到 S
例（课件经典） |  交错调和级数重排后和变为原来的 1/2

已知 ln2 = 1 − 1/2 + 1/3 − 1/4 + ⋯（交错调和级数的和）

重排为：1 − 1/2 − 1/4 + 1/3 − 1/6 − 1/8 + 1/5 − ⋯（每个奇数项后跟两个偶数项）

每组三项：1/(2n−1) − 1/(4n−2) − 1/(4n) = 1/(4n−2) − 1/(4n) = (1/2)[1/(2n−1) − 1/(2n)]

重排后级数 = (1/2) × 原级数 = ln2/2

仅改变加法顺序，和就从 ln2 变成了 ln2/2！这是条件收敛不满足加法交换律的直观例子。

7.2 级数乘法：Cauchy 乘积

Cauchy（对角线）乘积定义

类比多项式乘法，合并"斜对角"的项：

cₙ = a₀bₙ + a₁bₙ₋₁ + ⋯ + aₙb₀ = ∑k=0n aₖbₙ₋ₖ

称 ∑cₙ 为 ∑aₙ 与 ∑bₙ 的 Cauchy 乘积。

正方形乘积（辅助工具）

dₙ = ∑i,j≤n，max(i,j)=n aᵢbⱼ（第 n 个"L形框"所有项之和）

关键性质：Sₙ(d) = Sₙ(a)·Sₙ(b)，故两个级数收敛 → 正方形乘积收敛且等于两者之积。

7.3 四个定理（课件全部给出）

定理条件结论
定理 1 ∑aₙ 和 ∑bₙ 均绝对收敛 Cauchy 乘积 ∑cₙ 绝对收敛，且 ∑cₙ = (∑aₙ)(∑bₙ)
定理 2 ∑aₙ 和 ∑bₙ 均收敛 正方形乘积 ∑dₙ 收敛，且 ∑dₙ = (∑aₙ)(∑bₙ)
定理 3 ∑aₙ 绝对收敛，∑bₙ 收敛 Cauchy 乘积 ∑cₙ 收敛，且 ∑cₙ = (∑aₙ)(∑bₙ)
定理 4 ∑aₙ, ∑bₙ, ∑cₙ 三者均收敛 ∑cₙ = (∑aₙ)(∑bₙ)（证明在幂级数中给出）
注意：两个收敛级数的 Cauchy 乘积不一定收敛！

取 aₙ=bₙ=(−1)ⁿ/√(n+1)，两者均条件收敛，但可验证 |cₙ| ≥ 2(n+1)/(n+2) → 2 ≠ 0，Cauchy 乘积发散！

定理1"两绝对收敛"的条件是本质性的，不可弱化为"仅收敛"。

高频易错点（8条）

错误1

aₙ→0 就认为收敛。调和级数是终身反例。

错误2

两等价量相减直接用一阶等价。∑[1/n−ln(1+1/n)] 必须展到 1/(2n²)。

错误3

Raabe 括号里的分数方向写反。是 aₙ/aₙ₊₁（前/后），不是 aₙ₊₁/aₙ。

错误4

变号级数用等价替换。通项等价不代表级数敛散等价。

错误5

"几乎等于"代替严格证明。需用分离性取中间值才能严格。

错误6

条件收敛误以为可随意交换加法顺序。只有绝对收敛才行（更序定理）。

错误7

两收敛级数的 Cauchy 乘积一定收敛。不对，至少需要一个绝对收敛（定理3）。

错误8

绝对值换元后直接比较变号级数。∑sin(nx)/nᵖ 的条件收敛证明需要 sin²nx = (1−cos2nx)/2 技巧。
